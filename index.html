About this app

This is a calculator app that only requires the operation of one button to work.

  • Instructions for using the app

    • Hold the 'a' key to start a slow rightward scrolling selection over the buttons. Release 'a' to select that button.

  • Test results

    • I ran a test comparing using the app with my finger to my nose. In the nose condition, I held my keyboard up so I could still see the screen. Results: I found that using the app with my finger resulted in an average time of 35 seconds for a 6-button input sequence and caused no errors. With my nose, I found an average time of 49 seconds and one error.
